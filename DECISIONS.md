# Architecture Decisions

| Decision | Choice |
|---|---|
| Backend | Express on Render |
| Frontend | Next.js on Vercel |
| Admin | /admin inside Next.js (one repo) |
| Language | JavaScript only |
| Animations | Framer Motion + Tailwind |
| Dynamic pages | None (single page) |
| Contact | MongoDB + Resend email |
| Volunteering | DB + Admin CRUD |
| SVG Logos | Cloudinary |
| Admin theme | Dark/light toggle matching portfolio |
| Backend comms | REST API only |
